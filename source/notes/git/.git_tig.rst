Switching views
===============
* ``[m]ain``
* ``[s]tatus``
* ``[t]ree``
* ``[y] stash``
* ``[g]rep``
* ``[h]elp``

Status view
===========
* ``[u] Stage/unstage file or chunk``
* ``[!] Revert file or chunk``
* ``[C]ommit``
* ``[M]erge``
* ``[1] Stage line``
* ``[[] / []]     Increase/decrease the diff context``
