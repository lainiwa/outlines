Discrete values
===============

Bernoulli
---------
Two possible outcomes (0 or 1) and a single trial.

.. math::

  P(X = x) = \begin{cases}
  p & \text{if } x = 1 \\
  1 - p & \text{if } x = 0
  \end{cases}
  \qquad \text{where } x \in \{0, 1\}

.. math::

  E(X) = 1*p + 0*(1-p) = p

.. math::

  Variance(X) = E(X^2) – [E(X)]^2 = p – p^2 = p(1-p)
